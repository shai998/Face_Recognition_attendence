제 1 조 텐서 플로우를 활용한 얼굴인식 프로그램 (자동 출결 관리)

•개 요
   이세상에는 수많은 인공지능이있다. 우리는 우리의 일상에서 실질적으로 활용할수있는 우리에게 의미있는 인공지능을 만들고자 했다. 그러다가
우리가 비트 고급 과정을 수료하면서 가장 이슈화되었던 문제중 출석이라는 컨텐츠가 생각이났다.
   출석이라는 컨텐츠는 어떻게보면 굉장히 오래된 컨텐츠이고 이미 수많은 솔루션들이 나와있다. 그러나 지금까지 나온 솔루션들의 공통점은 모두
수동적으로 출석하는 방식이다.
   우리 프로그램은 카메라를 통해 자동으로 얼굴인식이 진행되고,  출석 처리가 되며 DB에 데이터가 전달된다. 만약 얼굴이 등록되어있지 않은 
누군가가 방문 시 간단한 알고리즘으로 제작된 챗봇이 방문한 사람에게 비트 에대한 간단한 정보를 제공해준다. DB에 저장된 데이터를 기반으로 
관리자는 관리자 클라이언트에서 실시간으로 학생들의 출입 및 출석기록을 확인할수있다. 또한 관리자 클라이언트 내 별도의 챗봇으로 (날짜, 이름)
검색을통한 특정 출결정보 확인이 가능하다.

•설계의 주안점
- 얼굴 인식을 통한 자동 출석 처리
- DB연동을 통한 실시간 출결정보 제공
- 관리자가 학생들의 출결 정보를 실시간으로 확인
- (날짜, 이름)검색을통한 특정 출결정보 확인
- 챗봇을 이용한 비트관련정보 제공

•응용 분야
- 물체 인식을 활용하는 모든 분야 (보안, 자율주행 자동차, 물류 등...)

•사용기술
- 개발 환경 : Windows 10 / .NET FrameWork 4.0
- 개발 도구 : C# / Visual Studio 2017 / WPF / Python 3.6 / Anaconda 3 / MS-SQL / PYQT5
- Python 패키지 : Tensorflow(1.13.1), Onnx(1.6.0), OpenCV(3.4.2), Dlib(19.19), Konlpy(0.5.2), Pygame(1.9.6)

•팀 원
- 김대정(팀장), 정찬묵, 조진수, 박승현, 최유진
