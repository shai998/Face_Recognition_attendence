제 1 조 얼굴 인식 모델을 활용한 학생 출결 관리 프로그램
•개 요
  최근 Tensorflow, Keras등 인공지능을 활용할 방법들이 다양해지면서 많은 인공지능 관련 프로그램들이 쏟아져나오고있다. 그래서 우리는 실생활에서 
활용할수있는 인공지능을 만들고자 하였다. 
  현재 우리는 학교를 다니면서 언제나 출석이라는 방식을 통해 우리가 학교에 왔음을 증명 및 관리를한다. 그리고 이 귀찮은 과정을 자동화 하기위해
지금까지도 지문인식, 전자출결등 많은 시도들이 있었다. 그러나 지금 까지의 출석의 형태는 학생들이 자신이 학교에 왔음을 알려주는 수동적인 방식
이다. 그리하여 우리는 얼굴 인식을 통한 자동 출결 확인 및 학생의 강의실내 위치여부를 메인 서버를 통해 DB에 저장하고 그 DB에 저장된 정보를
관리자 클라이언트를 통해 시각화하여 관리자에게 학생들의 출결 및 위치 여부 정보를 제공한다. 또한 챗봇 기능을 활용하여 모르는 사람이 강의실
방문 시 프로그램이 알아서 질의 응답을 통해 사람들에게 비트관련 정보를 제공한다.  
•설계의 주안점
- 얼굴 인식을 통한 출결 정보 업데이트
- 관리자가 학생들의 출결 정보를 실시간으로 확인
- (날짜, 이름)검색을통한 특정 출결정보 확인
- 챗봇을 이용한 비트관련정보 제공
- DB연동을 통한 실시간 출결정보 제공
•응용 분야
- 얼굴 인식을 활용하는 모든 분야
•사용기술
- 개발 환경 : Windows 10 / .NET FrameWork4.5
- 개발 도구 : C# / Visual Studio 2017 / WPF / Python / Anaconda3 / Python3.6 / MS-SQL / PYQT
- Python 패키지 : Tensorflow(1.13.1), Onnx(1.6.0), OpenCV(3.4.2), Dlib(19.19), Konlpy(0.5.2), Pygame(1.9.6)
•조 원
- 김대정, 정찬묵, 조진수, 박승현, 최유진
